Array

The type [n]T is an array of n values of type T.

The expression

var a [10]int

declares a variable a as an array of ten integers.

An array's length is part of its type, so arrays cannot be resized. This seems limiting, but don't worry; Go provides a convenient way of working with arrays.

 

Reference: https://tour.golang.org/moretypes/6

Reversal algorithm for array rotation

This program will rotate the array of size n by d elements.

Example:

Input :  arr[] = [1, 2, 3, 4, 5, 6, 7]
         d = 2
Output : arr[] = [3, 4, 5, 6, 7, 1, 2]

 

Below is the implementation of the problem:

 
package main
 
import "fmt"
 
// Function to left rotate arr of size n by d
func leftRotate(arr []int, d int) {
    if d == 0 {
        return
    }
    n := len(arr)
    reverseArray(arr, 0, d-1)
    reverseArray(arr, d, n-1)
    reverseArray(arr, 0, n-1)
 
}
 
// Function to reverse arr from index start to end
func reverseArray(arr []int, start int, end int) {
    for start < end {
        temp := arr[end]
        arr[end] = arr[start]
        arr[start] = temp
        start++
        end--
    }
}
 
// Function to print an array
func printArray(arr []int)  {
    n := len(arr)
    for i := 0; i < n; i++ {
        fmt.Printf("%x ", arr[i])
    }
}
 
//Driver program to test above functions
func main() {
    arr := []int{1, 2, 3, 4, 5, 6, 7}
    n := len(arr)
    d := 2
 
    // in case the rotating factor is
    // greater than array length
    d = d % n
    leftRotate(arr, d) // Rotate array by d
    printArray(arr)
}

Adding one to number represented as array of digits

Given a non-negative number represented as an array of digits, add 1 to the number (increment the number represented by the digits). The digits are stored such that the most significant digit is first element of array.

Examples :

Input : [1, 2, 4]
Output : [1, 2, 5]

Input : [9, 9, 9]
Output : [1, 0, 0, 0]

Below is the implementation to add 1 to number represented by digits.

package main

import "fmt"

// function to add 1 to the number
func addOne(s []int) []int{
    n := len(s)

    //add 1 to last digit and store the carry
    s[n-1] = s[n-1] + 1
    carry := s[n-1] / 10
    s[n-1] = s[n-1] % 10

    //loop from second last element
    for i := n - 2; i >= 0; i-- {
        if carry == 1 {
            s[i] = s[i] + 1
            carry = s[i] / 10
            s[i] = s[i] % 10
        }
    }

    if carry == 1 {
        s1 := []int{1}
        s1 = append(s1, s...)
        return  s1
    }
    return s
}

func main() {
    slice := []int{0, 7, 9, 9}
    s := addOne(slice)
    print(fmt.Sprintf("%v", s))
}

Write a program to calculate pow(x, n)

Examples :

Input : x = 2, n = 3
Output : 8

Input : x = 7, n = 2
Output : 49

package main

/* Function to calculate x raised to the power y */
func power(x int64, n int64) int64 {
	if n == 0 {
		return 1
	}
	if n%2 == 0 {
		return power(x, n/2) * power(x, n/2)
	} else {
		return x * power(x, n/2) * power(x, n/2)
	}
}

/* Program to test function power */
func main() {
	result := power(2, 5)
	println(result)
}

Time Complexity: O(n) Space Complexity: O(1)

The above code can be optimized to O(log n) by calculating power(x, y/2) once once. Please find the full code below:

package main

// Function to calculate x raised to the power n
func power(x int64, n int64) int64 {
	if n == 0 {
		return 1
	}
	temp := power(x, n/2)
	if n%2 == 0 {
		return temp * temp
	} else {
		return x * temp * temp
	}

}

func main() {
	result := power(2, 5)
	print(result)
}

Time Complexity: O(log n) Space Complexity: O(1)
